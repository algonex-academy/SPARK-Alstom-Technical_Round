1. Design a Ticket Management System.
Requirements:
You must create 3 tables:
users
tickets
departments, change any id datatype and use all DDL

Answer: 
create table users(user_id int primary key, 
  name varchar(100), 
  locality varchar(50), 
  age int, 
  gender varchar(10); 
  

create table tickets(ticket_id int primary key, 
  user_id int, 
  amount int, 
  ticket_date date
  train_id int
  coach_type varchar(50),
  seat_no int,
  dep_id int,
  foreign_key(dep_id,user_id);

create table departments(dep_id int primary key, 
  dep_name varchar(50));

2. Why using VARCHAR for ticket_id is a bad idea?
Answer: When we use VARCHAR on ticket_id, it may create the inconsistencies while retrieving the data and it may cause data intergity issue. While inserting the data, it may have the entries in incorrect format. 
Actually it is better when the values for ticket_id can be in specific pattern/format to be followed. 

3. Perform all following Operations
All DML Commands
Fetch Data (JOIN)
Tickets without Department
Users with Multiple Tickets
Answer: 
Applying DML Commands:
  1. create database new_project;
  2. alter table users add column address varchar(300);
  3. alter table tickets rename amount to ticket_price;
  4. drop departments;

Fetching Data using Joins
select u.user_id, u.name, t.ticket_id, t.amount, d.dep_name from users u 
join tickets t on u.user_id=t.user_id
join department d on t.dep_id=d.dep_id;

Tickets without department:
select t.ticket_id, d.dep_id from tickets join departments d on t.dep_id=d.dep_id where d.dep_id is null;

Users with Multiple Tickets:
select distinct t.user_id, u.name from tickets join users u on u.user_id=t.user_id where count(t.user_id)>1;

what is Lambda Function and list comprehension, give an example for each
Ans: Lambda function is a anonymous function in Python which is used when there is a single statement in the function, without defining the function name. It can be created by using lambda() and can be stored in a variable.
Ex: value=lambda(return num+100)
List Comprehension: It is used to create a list of values by specifying the algebraic expression by applying specific condition.
Ex: comp_list= [a*a for a in range(1,100)]
